# Pwm
Simpliest password manager that stores data in GPG-encrypted file.

## Usage
```sh
Usage:
  pwm.sh COMMAND ARG(S)
  Commands:
    e       - init/edit file
    l REGEX - list entries
```

## Examples
`pwm.sh l .` - print all entries
`pwm.sh l amazon` - print all entries for amazon
